# playlist-maker-android-YakushevichIlya
Учебный проект по созданию мобильного приложения с плейлистами

## Технологический стек
- Язык: Kotlin 1.9.0
- UI Framework: Jetpack Compose (Material 3)
- Управление состоянием: Coroutines & Flow
- Минимальная версия SDK (minSdk): 29
- Целевая версия SDK (targetSdk): 34
- Версия JDK: 17

## Требования для сборки
- Android Studio: Jellyfish / Koala (или новее).
- Gradle: 8.6+

## Как запустить проект
1. Склонируйте репозиторий.
2. Откройте проект в Android Studio.
3. Дождитесь завершения синхронизации Gradle.
4. Создайте эмулятор или подключите реальное устройство.
5. Запустите с помощью кнопки Run.
